* Intro

A backronym for an Emacs configuration called ENDURING:

Efficient
No-nonsense
Dependable
Utilitarian
Robust
Intuitive
Nifty
General-purpose

This backronym emphasizes the enduring qualities of your Emacs configuration, highlighting its efficiency, stability, and adaptability over the years.

This idea came from setting up an emacs newbie at work.  He was running emacs on a system that had emacs 23.1.1 (circa 2009) installed and also has access to 27.2

What if I could create a functional sane set of emacs defaults across all different versions, so for example he could learn the basic keybindings which would carry across each work project and also at home using the latest 29.1

The settings within this configuration are my own idea of emacs sane values which give the basic vanilla emacs the following notable features:

- dark theme by default
- tidied up interface to remove toolbar (kept menu bar and scroll bars)
- buffer / file completion in the minibuffer
- colourful buffer list
- saving versioned backup files into ~/backup
- programming indents to 4 spaces and no tabs N
- access to recently opened files
- saving desktop by default on exit
- winner-mode so can undo any window mistakes

and other bits and bobs (see comments below)

;; my/functions
(defun my/resize-window (delta &optional horizontal)
  "Resize window back and forth."
  (interactive)
  (let ((edge (if horizontal
                (car (window-edges))
                (car (cdr (window-edges))))))
    (if (= edge 0)
      (enlarge-window delta horizontal)
      (shrink-window delta horizontal))))

(defun my/comment-or-uncomment ()
  "Comments or uncomments the current line or region."
  (interactive)
  (if (region-active-p)
    (comment-or-uncomment-region
      (region-beginning)(region-end))
    (comment-or-uncomment-region
      (line-beginning-position)(line-end-position))))
